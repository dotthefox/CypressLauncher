# CypressLauncher

The Cypress Launcher is an easy to use launcher for [Cypress](https://github.com/BreakfastBrainz2/PvZCypress), a project that reimplements Dedicated Servers for the PvZ Shooters. 
This launcher is the continued work of [BreakfastBrainz2's](https://github.com/breakfastbrainz2) launcher used for the 2024 playtesting of Cypress. With this launcher comes a lot of documentation, which is recommended to read through thorougly in order to join and host servers.

# Building
The launcher should build properly long as you're using Visual Studio 2026 with .NET desktop development.
